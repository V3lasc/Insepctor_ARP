This program will help detect <a href='https://www.imperva.com/learn/application-security/arp-spoofing/' target='_blank'>ARP poisoning</a> attacks from a threat actor (attacker).

I would like to give credit to <a href='https://github.com/saintvic20/ARP-SPOOF-DETECTOR'>saintvic20</a> during my research on creating this tool.

<h2>Installation</h2>

```
git clone https://github.com/V3lasc/Inspector_ARP.git
cd Inspector_ARP
python3 Inspector_ARP.py
```
